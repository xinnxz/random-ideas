How It Works

The Science Behind Liquid Art

This animation uses a metaball algorithm implemented as a WebGL fragment shader. Multiple invisible spheres (blobs) float across the canvas. For every pixel on screen, the shader calculates the influence of each blob based on its distance.

When blobs get close enough, their influence fields overlap and merge — creating the characteristic "liquid merging" effect. The colors are generated dynamically using HSL color space, producing smooth rainbow-like gradients.

fragment.glsl 
float metaball(vec2 p, vec2 center, float r) {
    float d = distance(p, center);
    return (r * r) / (d * d);
}

// Sum influence of all blobs
float total = 0.0;
for(int i = 0; i < NUM_BLOBS; i++) {
    total += metaball(uv, blobs[i], radii[i]);
}

// Apply threshold → liquid surface
if(total > 1.0) {
    gl_FragColor = mix(color1, color2, total);
}
