# Random Ideas 💡

Hai! Ini adalah *repository playground* saya. 

Folder ini berisi kumpulan hasil eksplorasi, ide acak, dan latihan *front-end* harian (HTML, CSS, dan Javascript murni) yang saya buat untuk bersenang-senang dan belajar.

## Struktur Project

Setiap eksperimen atau ide disimpan di dalam foldernya masing-masing:

- `fanta-slider/` - Eksperimen membuat slider produk (seperti web Fanta).
- `liquid-animation/` - Latihan membuat animasi berefek cairan (liquid) menggunakan CSS/JS.
